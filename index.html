This Page displays the news feed from multiple API : Gaurdian API, News API and Civic Feed API. It displays 3 blocks of data which details on Summary of the headlines for last 7 days, next block is for count of articles per day and last one displays URLs.

Why We Did It

Given the abundance of information across the web Team Bravo has committed to the development of a web platform that allows interested parties, such as journalists and media analysts, to quickly and efficiently dissect the prevalence of a topic within the news cycle. The product aims to aggregate and analyze multiple high volume information outlets such as news portals and social media sites and present the user with an easy to digest overview. To best facilitate this, Team Bravo aims to utilize the dynamic 3D objects generation power of a-Frame. These objects will later have filters and further interactivity, beyond the initial spatial rotation
